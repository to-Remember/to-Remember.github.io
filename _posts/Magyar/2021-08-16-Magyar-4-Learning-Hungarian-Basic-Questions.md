# 기본 질문 모음   

1. **Kí vagy?**       
너 누구야?       
(vagy로 주어파악 가능 - Te)       

2. **Kí vannak?**    
그들은 누구에요? 혹은 당신들은 누구세요?       
(vannak으로 주어파악 가능 - Ök, Önök)       

3. **Hol éltek?**       
너네 어디살아?       
(éltek으로 주어파악 가능 - Ti)       

4. **Hany éves vagy?**       
너 몇살이야?       
(vagy로 주어파악 가능 - Te)       

5. **Milyen nemzetisegü vagy?**       
너 국적이 뭐야?       
(vagy로 주어파악 가능 - Te)       

6. **Milyen nyelven beszélsz?**       
어떤 언어 구사해?       
(vagy로 주어파악 가능 - Te)       

7. **Milyen nyelven tanulsz?**       
어떤 언어 공부해?       
(vagy로 주어파악 가능 - Te)       






오류가 있다면 알려주세요:]
